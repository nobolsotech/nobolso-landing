# nobolso-landing
Landing page do app No Bolso construída em Next.js e React.

## Descrição

No Bolso Waitlist é uma aplicação web construída em React e Next.js no formato SPA (Single Page Application).

O objetivo do site é apresentar o futuro app No Bolso e coletar dados de pessoas interessadas em participar da lista de espera para o lançamento oficial.
- Landing Page com apresentação do app.
- Formulário de inscrição para coleta de dados dos usuários interessados.
- Estrutura otimizada para lista de espera.
- Implementação simples e escalável usando Next.js.
